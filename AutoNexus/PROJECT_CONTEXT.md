\# AutoNexus



\## Overview

AutoNexus is a multi-vendor automobile spare parts marketplace for Cameroon.



It connects:

\- Car owners

\- Mechanics

\- Spare parts sellers

\- Administrators



\## Tech Stack



Frontend

\- React

\- JavaScript

\- Tailwind CSS



Backend

\- FastAPI (Python)



Database

\- MongoDB



Authentication

\- JWT

\- bcrypt



\## Project Goal



Build a secure, scalable, modern marketplace where users can search for vehicle spare parts, compare prices, connect with verified sellers, and order products easily.



\## Current Features



\- User authentication

\- Seller dashboard

\- Product management

\- Homepage

\- Product catalogue

\- Categories

\- Search

\- Admin dashboard
